* **Project Name:** 2D Oblique Projectile Motion Simulator
* **Description:** A physics-driven 2D Unity game that simulates oblique projectile launches across diverse environmental mediums (air, water, honey, vacuum) and planetary gravity fields. Players manipulate initial velocity, launch angle, and mass to strike targets. The system models physical behaviors in real time, leveraging both analytical solutions for linear viscous drag (derived from Stokes' Law) and numerical approximation via Euler's Method to dynamically calculate and render projectile trajectories.
* **Techniques:** Classical Mechanics (Kinematics), Ordinary Differential Equations (ODEs), Numerical Integration (Euler's Method), Fluid Dynamics (Viscous Drag), Trajectory Prediction, 2D Vector Mathematics.
